Connect to Supabase
First-time setup. You'll only do this once per device.
1
Go to your Supabase project → Settings → API. Copy your Project URL and anon / public key.
2
In the Supabase SQL Editor, run this once to create the data table:
create table if not exists app_state (
  id integer primary key default 1,
  state jsonb not null default '{}',
  updated_at timestamptz default now()
);
alter table app_state enable row level security;
create policy "team access" on app_state
  for all using (true) with check (true);
3
Paste your credentials below.
Your anon key is safe to share — it's a public key by design.
